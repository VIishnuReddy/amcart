# amcart
Spring Boot based e-commerce backend.

## Features implemented

- User registration
- Role-based access model
- Address mapping (OneToMany)
- BCrypt password hashing
- JPA auditing
- Global exception handling
- MySQL integration
- DTO validation

## Tech Stack

- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok

## Architecture

Controller → Service → Repository → Entity → DTO → Mapper
